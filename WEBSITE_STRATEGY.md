# QI Website Sitemap and UI/UX Recommendation

## 1) Sitemap

- About QI (`/index.html`)
- Consultation Services (`/consultation.html`)
- Audit Services (`/audit.html`)
- Training Services (`/training.html`)
- Contact Us (`/contact.html`)

## 2) Modern UI/UX Layout Recommendation

- Global header: QI logo + full navigation + mobile menu toggle.
- Hero section on each page: clear page title, short value statement, and primary CTA.
- Service cards: concise, scannable blocks for key offerings and benefits.
- Corporate visual direction: clean light surfaces, blue/teal brand palette, soft gradients, and subtle shadows.
- Typography: strong heading font for hierarchy and highly readable body font for professionalism.
- Conversion elements:
  - Training page includes `Sign Up` and `Certificate Verification`.
  - Contact page includes inquiry/sign-up form.
- Accessibility and responsiveness:
  - Keyboard-friendly controls
  - High contrast text
  - Mobile-first adaptive grids and touch-friendly buttons

## 3) Technical Delivery Included

- Multi-page responsive website
- Shared stylesheet and JavaScript for consistent design and behavior
- Mobile navigation, scroll reveal animations, modal verification flow, and form handling
